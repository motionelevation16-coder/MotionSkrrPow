> This is a big day. Keep it focused. Every checkbox here is something only *you* can do.

## 🟡 Phase 1 — Money & Accounts
**Time:** Morning | **Est. ~1 hour** | Do this first, everything depends on it.

- [ ] Check salary has arrived in account
- [ ] Decide domain budget — **suggested max: €20 total** to start (domain + nothing else)
- [ ] Buy domain for Motion Elevation (`motionelevation.de` or `.com` — .de preferred for DACH)
- [ ] Open AWS port 5678 — follow guide at `/projects/infrastructure/aws-port-guide.md` (10 min, just AWS console clicks)
- [ ] Log into ElevenLabs — check if 100k credits are usable without an active subscription (just try generating audio)

**✅ Phase 1 done = you have a domain, n8n is reachable, and you know your ElevenLabs status.**

## 🟠 Phase 2 — First Deployments
**Time:** Afternoon | **Est. ~2 hours** | Get something live before the meeting.

- [ ] Deploy Motion Elevation landing page — use **Carrd** (free tier, dead simple) or Framer if you prefer
- [ ] Connect your new domain to the landing page (Carrd has a guide, ~15 min)
- [ ] Set up Tally.so quiz for lead capture — free, just needs an email to sign up
- [ ] Test the full visitor flow: landing page → quiz → email captured ✅

> **Why this matters:** When you show cousin the meeting, you're not showing a design file — you're showing a live URL. That changes the whole conversation.

**✅ Phase 2 done = Motion Elevation is live on a real domain with a working lead capture.**

---

## 🔵 Phase 3 — Cousin Meeting
**Time:** Whenever you meet | **Est. ~1–1.5 hours** | This is the strategy session.

- [ ] Share `COUSIN-BRIEF.md` with cousin before or at the start of the call
- [ ] Walk him through each project (5 min each — don't go deep, let him ask)
- [ ] Ask him directly: *"What can you build or help with fastest?"*
- [ ] Pick **1 joint project** to start together — API listing (fastest to money) or video pipeline (most fun)
- [ ] Set a next sync date before you hang up — don't leave without a concrete follow-up

> **Suggested priority for joint project:** HookScore or Prediction Market API → RapidAPI listing. Fastest path to actual revenue with his help.

**✅ Phase 3 done = you have a co-builder and a clear next step.**

## 🟢 Phase 4 — First Outreach
**Time:** Evening | **Est. ~30–45 min** | Optional, but do it if you have energy left.
