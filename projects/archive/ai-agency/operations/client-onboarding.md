# ✅ Client Onboarding Checklist

*Step-by-step guide to getting a new client live in 1 week*

---

## Day 1: Kickoff Call (30 minutes)

### Before the Call
- [ ] Send calendar invite with Zoom link
- [ ] Prepare intake questionnaire
- [ ] Have demo agent ready to reference

### During the Call
- [ ] Introduce yourself and the process
- [ ] Collect practice information:
  - [ ] Practice name (exact spelling)
  - [ ] Full address
  - [ ] Main phone number
  - [ ] Website URL
  - [ ] Hours of operation (by day)
  - [ ] List of dentists (names, specialties)
  - [ ] Services offered
  - [ ] Insurance plans accepted
  - [ ] Emergency protocol (after-hours number?)
  
- [ ] Understand their specific needs:
  - [ ] Primary goal (new patients? fewer missed calls? after-hours?)
  - [ ] What questions do they get most often?
  - [ ] Any special scripts or phrases they use?
  - [ ] What should trigger a transfer to human?
  
- [ ] Technical setup:
  - [ ] What phone system do they use?
  - [ ] What practice management software? (Dentrix, Eaglesoft, etc.)
  - [ ] What calendar? (Google, Outlook, built-in?)
  - [ ] Who should receive text notifications?

- [ ] Set expectations:
  - [ ] Timeline: Live within 5-7 days
  - [ ] Trial period: 2 weeks to adjust
  - [ ] Support: How to reach you for changes

### After the Call
- [ ] Send summary email with what you collected
- [ ] Get their approval on the information
- [ ] Request calendar access / API credentials

---

## Day 2-3: Build the Agent

### Configuration
- [ ] Create new agent in platform (Bland/Vapi/etc.)
- [ ] Input all practice information
- [ ] Customize the system prompt
- [ ] Set up voice (select appropriate voice)
- [ ] Configure call handling rules

### Integrations
- [ ] Connect calendar
- [ ] Set up appointment booking flow
- [ ] Configure SMS confirmations
- [ ] Set up notification system (staff alerts)
- [ ] Create call logging (Airtable/Sheets)

### Testing
- [ ] Internal test calls (5-10 scenarios):
  - [ ] New patient booking
  - [ ] Existing patient rescheduling
  - [ ] Insurance question
  - [ ] Hours/location question
  - [ ] Emergency scenario
  - [ ] Request to speak with human
- [ ] Fix any issues found
- [ ] Document any edge cases

---

## Day 4: Client Review Call (30 minutes)

### Before the Call
- [ ] Have agent live in test mode
- [ ] Prepare list of test scenarios
- [ ] Have recording examples ready

### During the Call
- [ ] Walk through the agent setup
- [ ] Do live test calls together
- [ ] Get feedback on:
  - [ ] Voice/tone — does it match their brand?
  - [ ] Responses — anything to adjust?
  - [ ] Booking flow — is it correct?
- [ ] Note all requested changes

### After the Call
- [ ] Send summary of requested changes
- [ ] Implement changes within 24 hours
- [ ] Send confirmation when done

---

## Day 5-6: Implement Changes & Final Testing

### Changes
- [ ] Make all requested adjustments
- [ ] Re-test affected scenarios
- [ ] Document final configuration

### Final Checklist
- [ ] All information accurate
- [ ] Calendar integration working
- [ ] SMS notifications working
- [ ] Transfer to human working
- [ ] Call logging working
- [ ] 10+ successful test calls completed

---

## Day 7: Go Live

### Preparation
- [ ] Schedule go-live call (15 minutes)
- [ ] Prepare phone forwarding instructions
- [ ] Have support contact ready

### Go-Live Call
- [ ] Confirm everything is ready
- [ ] Walk them through phone forwarding setup
- [ ] Do a live test call (they call in)
- [ ] Verify everything works end-to-end
- [ ] Set up monitoring for first 24 hours

### After Go-Live
- [ ] Send "You're Live!" email with:
  - [ ] How to access dashboard
  - [ ] How to request changes
  - [ ] Your support contact
- [ ] Monitor calls for first 24-48 hours
- [ ] Check in via text/email next day

---

## Week 2: Optimization

### Daily (first week)
- [ ] Review all calls from previous day
- [ ] Note any issues or improvement opportunities
- [ ] Make small adjustments as needed

### End of Week 1 Call (15 minutes)
- [ ] Review call volume and outcomes
- [ ] Discuss any issues they've noticed
- [ ] Get feedback from their team
- [ ] Make any necessary adjustments

### End of Trial (Week 2)
- [ ] Review performance metrics:
  - [ ] Total calls handled
  - [ ] Appointments booked
  - [ ] Calls transferred
  - [ ] Missed calls (should be zero)
- [ ] Present ROI summary
- [ ] Confirm they're continuing
- [ ] Process first invoice

---

## Ongoing (Monthly)

### Week 1 of Each Month
- [ ] Send monthly report:
  - [ ] Call volume
  - [ ] Appointments booked
  - [ ] Common questions (insights)
  - [ ] Any issues noted
- [ ] Offer optimization suggestions

### As Needed
- [ ] Respond to change requests (within 24 hours)
- [ ] Handle any escalated calls
- [ ] Update agent for holidays, schedule changes

### Quarterly (Optional for higher tiers)
- [ ] Strategy call
- [ ] Review performance trends
- [ ] Discuss expansion opportunities
- [ ] Get testimonial/referral
