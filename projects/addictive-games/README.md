# 💎 Addictive Games Project

*Building matching/idle/clicker games that make players lose track of time*

---

## Current Build: Crystal Match

A match-3 gem game with casino-level dopamine engineering.

### Features
- 🎮 Match-3 mechanics (swipe/click)
- ✨ Particle explosions on matches
- 📳 Screen shake on big combos
- 🎵 Satisfying sound effects
- 💰 Score multipliers & streaks
- 🎰 Casino-style feedback loops
- 📱 Mobile-responsive

### Tech Stack
- Pure HTML5/CSS/JavaScript
- No dependencies
- Runs in any browser

### Play
Open `game/index.html` in a browser.
