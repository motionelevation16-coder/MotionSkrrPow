## Objections (Legal Specific)

**"AI can't do legal work"**
→ "You're right—AI doesn't practice law. It handles the admin so YOU can practice more law. Document drafting, scheduling, billing—that's not lawyering, that's overhead."

**"What about confidentiality?"**
→ "We take privilege seriously. All data is encrypted, access-controlled, and never used to train models. Happy to sign an NDA and walk through our security."

**"My practice management software does this"**
→ "Most PMS handles the basics. We connect everything—your PMS, email, calendar, billing—so data flows automatically instead of being re-entered."

**"My clients expect personal service"**
→ "Absolutely. Automation handles the routine so you have MORE time for high-touch service. Clients notice when you're responsive and prepared—automation makes that possible."
