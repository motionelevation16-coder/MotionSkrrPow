## CARRD SETUP INSTRUCTIONS

### 1. Create Account
- Go to carrd.co
- Sign up (free tier = 3 sites)

### 2. Choose Template
- Pick "Blank" or "Profile" template
- Delete default content

### 3. Build Sections
- Add Container for each section
- Add Text elements
- Add Button (link to Calendly/Cal.com)
- Add Icons or emojis as visual breaks

### 4. Style
- Font: Clean sans-serif (Inter, DM Sans)
- Colors: Dark text, light background, one accent color
- Spacing: Generous (don't crowd)

### 5. Connect Domain (optional)
- Buy domain (~$10/year)
- Add to Carrd (free on paid plan, $19/year)

### 6. Publish
- Click Publish
- Share link everywhere
