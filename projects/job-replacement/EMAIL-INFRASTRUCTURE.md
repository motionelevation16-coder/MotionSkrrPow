## Domain Structure

### Why Multiple Domains?

- **Main domain** (motionelevation.com) = website, client emails, reputation
- **Outreach domains** = cold email, can be burned/replaced if flagged
- **Never risk your main domain on cold outreach**

### Recommended Setup

| Domain | Purpose | Email |
|--------|---------|-------|
| `motionelevation.com` | Main website, client comms | hello@, support@ |
| `getmotionelevation.com` | Cold outreach #1 | lyubo@, team@ |
| `motionelevation.co` | Cold outreach #2 | hi@, contact@ |
| `trymotionelevation.com` | Cold outreach #3 | hello@, team@ |
| `motionelevation.io` | Backup/future | — |

**Total cost:** ~$40-50/year for 4 domains

## Email Setup (Free Option)

### Zoho Mail (Free for 5 users)

1. Go to [zoho.com/mail](https://www.zoho.com/mail/)
2. Sign up for free plan
3. Add domain → Verify via DNS
4. Create mailboxes:
   - `lyubo@getmotionelevation.com`
   - `team@getmotionelevation.com`
   - etc.

### DNS Records Required

```
MX Records:
  mx.zoho.eu (Priority 10)
  mx2.zoho.eu (Priority 20)
  mx3.zoho.eu (Priority 50)

SPF (TXT):
  v=spf1 include:zoho.eu ~all

DKIM (TXT):
  [Generated by Zoho]

DMARC (TXT):
  v=DMARC1; p=none; rua=mailto:dmarc@yourdomain.com
```

## Domain Warmup (CRITICAL)

### Why Warm Up?

New domains have no reputation. Sending 100 emails day 1 = spam folder.

### Warmup Process

**Week 1-2:** Use a warmup tool
- Sends/receives emails automatically
- Builds positive engagement signals
- Opens, replies, marks as "not spam"

### Warmup Tools

| Tool | Cost | Notes |
|------|------|-------|
| **Instantly.ai** | Included in $37/mo plan | Best value |
| **Warmup Inbox** | $9/mo per inbox | Standalone |
| **Lemwarm** | Included with Lemlist | If using Lemlist |
| **Mailwarm** | $9/mo per inbox | Simple |

### Warmup Schedule

| Day | Emails Sent | Notes |
|-----|-------------|-------|
| 1-3 | 5-10/day | Warmup tool only |
| 4-7 | 10-20/day | Warmup only |
| 8-14 | 20-30/day | Warmup + few manual |
| 15-21 | 30-50/day | Start cold outreach (20/day) |
| 22-30 | 50-80/day | Scale to 50/day |
| 30+ | 80-100/day | Full volume |

**Timeline:** 3-4 weeks before full-scale sending

## Inbox Management

### Folder Structure (per inbox)

```
Inbox
├── Interested (manual label)
├── Meeting Booked (auto-label)
├── Not Interested (auto-label)
├── Out of Office (auto-label)
└── Bounced (auto-label)
```

### Response Handling

| Response Type | Action |
|---------------|--------|
| Interested | Reply personally, book call |
| Maybe later | Add to 30-day follow-up |
| Not interested | Remove from sequence, polite close |
| Out of office | Pause, resume when back |
| Bounce | Remove, clean list |
| No response | Continue sequence |

## Daily Sending Limits

### Safe Limits (per inbox)

| Provider | Daily Limit | Safe Volume |
|----------|-------------|-------------|
| Google Workspace | 500/day | 80-100/day |
| Zoho | 500/day | 50-80/day |
| Outlook/365 | 300/day | 50-80/day |

### Scale Strategy

```
5 inboxes × 80 emails/day = 400 emails/day
400 emails × 30 days = 12,000 emails/month
12,000 × 3% reply rate = 360 replies
360 × 30% book = 108 calls/month
108 × 25% close = 27 clients/month
```
