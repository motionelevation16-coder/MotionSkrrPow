## 🎯 How to Use This Package

**Week 1 — Get the site live:**
1. Push `index.html` to a GitHub repo → enable GitHub Pages
2. Point a domain (e.g. `motion-elevation.com`) at it
3. Update the mailto link to your real email

**Week 1–2 — Start outreach:**
1. Pick Sequence A, B, or C based on who you're targeting
2. Find 20–30 leads (Google Maps, LinkedIn, Instagram)
3. Send Email 1, log in the tracker
4. Day 3: Email 2. Day 7: Email 3

**Week 2+ — Book calls:**
1. Use the sales call script — don't memorize, just internalize
2. Run your n8n demo during calls
3. Close with Starter or Growth plan

**Ongoing — Test and improve:**
1. Use the subject line bank for A/B tests
2. Track what works in the outreach tracker
3. Double down on whatever sequence gets the best reply rate
